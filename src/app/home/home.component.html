Marvel

Projeto

Este projeto foi feito com o intuito de acessar uma Api Rest E exibir os dados dela. Os dados exibidos nesta aplicação são os personagens 'characters' As histórias 'comics' e os filmes 'series'.

Personagens

A pagina personagens fo feita com o intuito de mostrar alguns dos personagens listados na Api. Exibir Seu nome, e Sua descrição, porém muitos não possuem imagens nem descrição mas optei por exibli-los da mesma forma.

Histórias

As histórias apenas foram listadas. A ideia era fazer todas as paginas parecerem uma galeria A pagina de histórias em especifico fiz pensando em fazer quase uma loja de compras para cada livro.

Filmes

A pegada dos filmes foi bem similar com a a pegada das histórias, fiz um mural parecido com uma galeria para o usuário escolher qual deseja. Apesar de que nenhum vai para o site da compra xD
